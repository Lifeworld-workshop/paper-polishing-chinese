本文件提供中文学术论文的表达习惯规范，包括人称、语态、"本文"用法等。

## 1. 人称使用规范

### 基本规则
- **禁止使用**：我、我们、你、你们、咱们
- **推荐使用**：本文 / 本研究 / 笔者
- **例外**：在致谢部分可以使用"笔者"或"作者"

### 常见违规与修改

| 违规表达 | 修改方案 |
|----------|---------|
| 我们发现 | 研究发现 / 本文发现 |
| 我认为 | 本文认为 / 笔者认为 |
| 我们使用 | 本文采用 / 本文使用 |
| 我们收集了数据 | 本文收集了… / 数据来源于… |
| 让我们来看 | 下面考察 / 接下来分析 |
| 我们可以看到 | 可以发现 / 由表 X 可知 |

### 注意事项
- 经济学中文期刊（如《经济研究》）通常接受"本文"和"笔者"
- 部分期刊接受"我们"用于合著论文，但应保持全文一致
- 摘要中尽量避免"本文"，改用无主语句或以研究对象作主语

## 2. "本文"与"本研究"用法

### 适用场景

| 表达 | 适用场景 |
|------|---------|
| 本文 | 引言、贡献阐述、方法说明 |
| 本研究 | 偏重实证过程的叙述 |
| 本章 | 各章节开头或总结 |

### 使用频率控制
- **避免在同一段落中重复使用超过 2 次"本文"**
- 可用无主语句替代：
  - "本文使用了…"→"使用…数据"（承前省略主语）
  - "本文认为…"→"上述分析表明…"

### "本文"在摘要中的处理
- 部分高校学位论文规范要求摘要中不出现"本文""本研究"
- 替代方案：以研究问题或研究对象为主语
  - ❌ "本文研究了教育回报率的性别差异。"
  - ✅ "教育回报率的性别差异是劳动经济学的重要议题。"

## 3. 主动语态与被动语态

### 基本原则
中文学术写作**以主动语态为主**，被动语态仅在特定场景使用。

### 适用被动语态的场景

| 场景 | 示例 |
|------|------|
| 强调动作承受者 | "该政策被广泛采纳。" |
| 施动者不明或不重要 | "数据由国家统计局发布。" |
| 客观化表述 | "上述假说可以通过实证检验加以验证。" |

### 不宜使用被动语态的场景

| 场景 | 不当表达 | 修改 |
|------|---------|------|
| 研究行为 | "回归分析被进行了" | "进行了回归分析" |
| 普通叙述 | "结果被发现是显著的" | "结果显著" |
| 论点陈述 | "这一观点被本文支持" | "本文支持这一观点" |

### 中文"被"字句注意
- 中文"被"字句常含被动/不利含义（"被批评""被淘汰"）
- 学术写作中可用"由""经""受""为…所"替代"被"
  - "被验证为有效的"→"经验证有效"
  - "被广泛引用"→"为学界广泛引用"

## 4. 时态规范（中文特有）

中文无严格时态变化，但通过时间状语和助词体现时间关系：

| 论文部分 | 时间表达 | 示例 |
|----------|---------|------|
| 引言-背景 | 近年来 / 长期以来 | "近年来，数字经济发展迅速。" |
| 文献综述 | 已有研究 / 前人研究 | "张三（2020）研究了…" |
| 理论分析 | 无时间标记（一般现在） | "假设经济中存在两类工人。" |
| 数据说明 | 过去时标记 | "本文使用了 2010—2020 年的数据。" |
| 实证结果 | 发现/表明/显示 | "表 3 的结果表明…" |
| 结论 | 本文…了 / 研究发现 | "本文分析了…的影响机制。" |

## 5. 数字与单位规范

依据 GB/T 15835-2011《出版物上数字用法》：

### 使用阿拉伯数字
- 统计数据、精确数值：5.2%、1 234 人、3.14
- 年份、日期：2020 年、5 月 15 日
- 表格序号、公式编号：表 1、式（3）

### 使用汉字数字
- 概数：三四十个、几百、数千
- 固定用法：第一、第二（序数词）
- 成语：三令五申、四面八方
- 星期：星期一

### 数值规范
- 四位及以上数字使用半角空格分节：12 345（非 12,345）
- 百分比：百分号前不空格，如 5.2%
- 负数：用"−"（减号），非"-"（连字符）

## 6. 引用与注释规范

### 正文引用格式
- 单一作者：张三（2020）指出… / （张三，2020）
- 两位作者：张三和李四（2020） / （张三和李四，2020）
- 三位及以上：张三等（2020） / （张三等，2020）
- 外文文献：Acemoglu（2002） / （Acemoglu，2002）

### 注意事项
- 引用序号放在标点符号之前（部分期刊要求之后，需查阅具体规范）
- 同一处引用多篇文献用分号分隔：（张三，2018；李四，2020）
- 年份按时间排列

## 7. 中英文混排规范

### 空格规则
- 中文与英文/数字之间加半角空格
  - ✅ "GDP 增长率为 6.5%"
  - ❌ "GDP增长率为6.5%"
- 英文与中文标点之间不加空格
  - ✅ "Acemoglu（2002）指出，"
  - ❌ "Acemoglu （2002） 指出 ，"

### 英文术语首次出现规则
- 首次出现须提供中文全称，括号内标注英文：
  - "全要素生产率（Total Factor Productivity，TFP）"
- 后续可直接使用缩写：TFP

## 8. 各章节表达规范

### 摘要
- 200—300 字为宜
- 结构：背景-问题-方法-发现-贡献
- 避免出现引用文献、公式、表格编号

### 引言
- 开篇阐述研究背景与意义
- 清晰提出研究问题
- 概述研究方法与数据
- 预告主要发现
- 说明创新与贡献
- 预告全文结构

### 文献综述
- 按主题/流派/时间组织，非简单罗列
- 每段末尾指出现有研究不足
- 最后一段说明本文如何填补空白

### 结论
- 概括主要发现（2—3 句）
- 阐述政策含义
- 说明局限性
- 指出未来研究方向
